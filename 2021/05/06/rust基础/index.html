    rust函数

    fn <函数名> ( <参数> ) <函数体>

    Rust 中定义函数如果需要具备参数必须声明参数名称和类型：

    fn main() {
    another_function(5, 6);
}

fn another_function(x: i32, y: i32) {
    println!("x 的值为 : {}", x);
    println!("y 的值为 : {}", y);
}

    在参数声明之后用 -> 来声明函数返回值的类型（不是 : ）

    如果没有明确声明函数返回值的类型，函数将被认为是”纯过程”，不允许产生返回值，return 后面不能有返回值表达式。这样做的目的是为了让公开的函数能够形成可见的公报

    fn main() {
    fn five() -> i32 {
        5
    }
    println!("five() 的值为: {}", five());
}

    fn main() {
    let number = 3;
    if number < 5 {
        println!("条件为 true");
    } else {
        println!("条件为 false");
    }
}

    Rust 中的条件表达式必须是 bool 类型

    fn main() {

    if number {   // 报错，expected `bool`, found integerrustc(E0308)

    while

    let mut number = 1;
    while number != 4 {
        println!("{}", number);
        number += 1;
    }
    println!("EXIT");
}

    let mut i = 0;
while i < 10 {
    // 循环体

    fn main() {
    let a = [10, 20, 30, 40, 50];
    for i in a.iter() {
        println!("值为 : {}", i);
    }
}
// for循环，遍历数组；a.iter() 代表 a 的迭代器（iterator）

    在 C 语言中 for 循环使用三元语句控制循环，但是 Rust 中没有这种用法，需要用 while 循环来代替

    fn main() {
    let s = ['R', 'U', 'N', 'O', 'O', 'B'];
    let mut i = 0;
    loop {
        let ch = s[i];
        if ch == 'O' {
            break;
        }
        println!("\'{}\'", ch);
        i += 1;
    }
}
// loop循环;开头和结尾无法判断是否继续进行循环

    // break可以把值返还，类似return；
fn main() {
    let s = ['R', 'U', 'N', 'O', 'O', 'B'];
    let mut i = 0;
    let location = loop {
        let ch = s[i];
        if ch == 'O' {
            break i;
        }
        i += 1;
    };
    println!(" \'O\' 的索引为 {}", location);
}

    所有权

    高效使用内存

    • Rust 中的每个值都有一个变量，称为其所有者。
    • 一次只能有一个所有者。
    • 当所有者不在程序运行范围时，该值将被删除。

    变量和数据交互

    移动I–基本数据类型【仅在栈中

    let x = 5;
let y = x;

    移动II–发生

    let s1 = String::from("hello");
let s2 = s1; 
println!("{}, world!", s1); // 错误！s1 已经失效rust

    克隆

    fn main() {
    let s1 = String::from("hello");
    let s2 = s1.clone();
    println!("s1 = {}, s2 = {}", s1, s2);
}

    引用和租借_一种指针

    fn main() {
    let s1 = String::from("hello");
    let s2 = &s1;
    println!("s1 is {}, s2 is {}", s1, s2);
}

    fn main() {
    let s1 = String::from("hello");

    let len = calculate_length(&s1);

    println!("The length of '{}' is {}.", s1, len);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}

    租借一般不能修改原值；除非原值定义为可变变量

    fn main() {
    let mut s1 = String::from("run");
    // s1 是可变的

    let s2 = &mut s1;
    // s2 是可变的引用

    s2.push_str("oob");
    println!("{}", s2);
}

    但是可变引用不允许多重引用

    let mut s = String::from("hello");

let r1 = &mut s;
let r2 = &mut s;

println!("{}, {}", r1, r2);

    不允许悬垂引用

    fn main() {
    let reference_to_nothing = dangle();
}

fn dangle() -> &String {
    let s = String::from("hello");

    &s
}
