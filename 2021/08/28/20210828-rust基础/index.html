    常见编程概念

    变量和可变性

    let:变量默认是不可改变的（immutable）;变量名之前加 mut 来使其可变，但类型不能改变；但同时可以多次使用let，实现变量的隐藏（shadowing）

    const:常量总是不能变，必须注明值的类型；在单词之间使用全大写加下划线

    数据类型

    静态语言：编译时知道所有变量的类型；

    标量类型scalar
    整型 i/u 8/16/32/64```
    浮点型 f 32/64
    布尔型 true/false
    字符类型 char2

    元组，长度固定；可以模式匹配；可以用[.索引]来访问 let tup: (i32, f64, u8) = (500, 6.4, 1);

    控制流

    if…else if…else if
    if 表达式可以在let语句右侧使用，每个分支要为同一个类型
    loop/while/for
    loop/continue/break 循环标签，如下面的’counting_up
    while/loop if break/
    for 遍历集合中元素 
    fn main() {
    let mut count = 0;
    'counting_up: loop {
        println!("count = {}", count);
        let mut remaining = 10;

        loop {
            println!("remaining = {}", remaining);
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up;
            }
            remaining -= 1;
        }

        count += 1;
    }
    println!("End count = {}", count);
}

    所有权——关于内存管理

    1. Rust 中的每一个值都有一个被称为其 所有者（owner）的变量。
    2. 值在任一时刻有且只有一个所有者。
    3. 当所有者（变量）离开作用域，这个值将被丢弃。

    第二个字符串类型:String。这个类型被分配到堆上，所以能够存储在编译时未知大小的文本

    let s = String::from(“hello”);

    let mut s = String::from("hello");
s.push_str(", world!"); // push_str() 在字符串后追加字面值
println!("{}", s); // 将打印 `hello, world!`

    几种变量与数据交互的方式，以String为例

    移动

    let s1 = String::from("hello");
let s2 = s1;

println!("{}, world!", s1);

// 得到错误，因为s2只是浅拷贝，由rust的回收机制，如果s1和s2都被保留，那么会收时会发生二次回收问题。因此rust使得s1失效Rust 永远也不会自动创建数据的 “深拷贝”。
但是整型没有深浅拷贝的区别。
