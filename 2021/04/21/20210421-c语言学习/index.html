    基本数据类型

    1.整形数据

    基本型 int unsigned int
    短整型 short int ; short unsigned short
    长整型（整数后加L） long int ； long unsigned long
    十进制整数
    八进制整数 整数开头加数字符0
    十六进制整数 整数开头加0x/0X

    2.浮点型数据

    float double long double

    3.字符型数据

    普通字符

    转义字符常量
    \n 换行符 \ 转义字符
    \t 制表符

    字符串

    一个字符型数据可以用字符格式%c和整形格式%d输出；

    一个字符型变量只可以接受一个字符型常量，但不能接受一个字符串；

    char c1=’a’ ✔；char c1=”a“❌

    输入和输出基础

    单个字符输出和输入

    putchar()输出——-getchar()读取———putchar(getchar())

    格式输出和输入

    printf()—-printf(输出格式控制字符串，输出项表)

    scanf()—-scanf(输入格式控制字符串，数据存储地址项表)

    数据运算

    赋值运算

    x +=5 ；x*=u+v

    ++/– i ++/–；有四种组合方式

    逻辑运算

    &&和 ||或 ！非
