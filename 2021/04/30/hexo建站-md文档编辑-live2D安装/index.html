    hexo建站+md文档编辑+live2D安装

    建站 工具 hexo live-2D

    字数:257 阅读 :

      1. 参考网站

    工具篇

    github账号+仓库
    Git https://git-scm.com/download 分布式版本控制系统
    Node https://nodejs.org/en JavaScript 环境 node/npm 命令
    Hexo npm install -g hexo -g 参数表示在全局进行安装 hexo 命令

    安装篇

    hexo init npm install hexo init npm install
    以上三者都使用git bash， 可以将某文件夹初始化为 Hexo 根目录
    _config.yml 用来修改网站配置文件；
    hexo s 启动本地站点 http://localhost:4000/

    远程登录篇

    $ git config –global user.name your_name
    $ git config –global user.email your_email
    $ npm install hexo-deployer-git –save _config.yml内需要修改配置 branch：要把master改为main
    ssh-keygen -t rsa -C “your_email” 配置 SSH；/c/Users/user/.ssh/id_rsa.pub 复制到Github的setting里面的SSH and GPG keys
    ssh -T git@github.com 验证是否添加成功

    其他配件包

    npm install hexo-asset-image –save config修改：post_asset_folder: true
